## [2.1.0] - 2026-04-03

### Added
- **Neo4j 图数据库支持**: 新增完整的图数据库集成，支持 AI 聊天中的智能章节分析
- **成本追踪系统**: 新增全面的令牌使用监控和成本计算功能
- **AI 聊天增强**: 新增完整 AI 聊天数据库模式与流式对话功能
- **角色进化跟踪**: 集成角色一致性跟踪和相似度检测功能
- **智能重试机制**: 新增 QwenClient 可靠性优化和智能重试
- **AI 增强型 E2E 测试框架**: 新增 AI 驱动的端到端测试框架及相关组件
- **上下文预加载**: 增强上下文预加载与统一管理功能
- **手动迁移脚本**: 切换至手动迁移脚本系统并增强数据库 Schema 设计
- **统一日志系统**: 统一配置后端及 Worker 日志系统

### Changed
- **优化 Neo4j 生产环境部署**: 提升安全性和配置优化
- **优化 QwenClient**: 支持 max_tokens 可选参数且新增小说创作管理器
- **优化 JSON 提取**: 改进 JSON 提取和模型调用的 token 限制处理
- **优化伏笔同步**: 修复伏笔同步失败问题

### Fixed
- **伏笔同步**: 修复伏笔同步失败问题
- **Token 限制处理**: 优化模型调用的 token 限制处理
