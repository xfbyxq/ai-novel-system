### 1. 稳定的选择器策略
```python
# 使用内容定位而非硬编码属性
"create_button": "button:has-text('创建小说')"
"novel_cards": ".ant-table-tbody tr[data-row-key]"
```

### 2. 智能等待机制
```python
def wait_for_novels_loaded(self, timeout: int = 10000):
    """等待小说列表加载完成"""
    self.wait_for_element_visible(self.SELECTORS["novel_cards"], timeout=timeout)
```
